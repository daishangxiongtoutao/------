Installation

32-bit MPANT Software

Install the 32-bit Software for Windows XP / Vista / 7 / 8 / 10. If you have already got an older version of the MPANT software for your device, please first uninstall the old version using the Start menu or Control panel.

Native 64-bit MPANT Software

Install the native 64-bit Software for Windows 7 / 8 / 10. If you have already got an older version of the 64-bit MPANT software for your device, please first uninstall the old version using the Start menu or Control panel.

MCS8 DLL Software

Unzip the MCS8 DLL Software into a directory of your choice.
